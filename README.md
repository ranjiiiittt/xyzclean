# xyzclean

Next.js landing page for an Australia-based cleaning company.

## Run locally

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`

## Project structure

- `app/` - app router pages, metadata, icon, and global styles
- `components/` - interactive UI pieces like the header and quote form
- `public/` - brand mark and custom SVG artwork used across the site
